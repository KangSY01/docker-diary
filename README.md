# 🗒️ Docker Diary

Docker + Nginx 로드밸런서 기반 고가용성 온라인 다이어리 서비스

## 📌 프로젝트 소개
사용자가 일기를 작성, 조회, 삭제하고 사진을 첨부할 수 있는 온라인 다이어리 웹 서비스.
Docker 컨테이너로 여러 개의 서버 인스턴스를 구성하고 Nginx를 로드밸런서로 활용하여
특정 서버에 장애가 발생하더라도 서비스가 중단되지 않는 고가용성 구조를 구현한다.

## 🛠️ 기술 스택
- **웹 서버**: Python + Flask
- **로드밸런서**: Nginx
- **데이터베이스**: PostgreSQL
- **컨테이너**: Docker + Docker Compose

## 📐 시스템 구조
사용자 → Nginx (로드밸런서) → Flask 서버1 / 서버2 / 서버3 → PostgreSQL
